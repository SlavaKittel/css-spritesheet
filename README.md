# CSS Spritesheet

A simple project demonstrating how to create and use a CSS spritesheet to optimize web performance by reducing HTTP requests.

# Overview

CSS spritesheets combine multiple images into a single image file and use CSS background positioning to display individual icons or graphics.  
This technique improves loading speed and performance, especially for icon-heavy interfaces.

This repository contains:
- A spritesheet image
- CSS styles for accessing individual sprites
- Example usage in HTML
- Display real-time FPS via a HUD
